Dream Fund
Dream Fund is a decentralized crowdfunding platform built with Next.js, React, and Ethereum smart contracts. Users can create fundraising campaigns, donate ETH to support causes, and track campaign progressâ€”all on the blockchain. The project leverages ethers.js for blockchain interactions and Web3Modal for wallet connectivity, providing a seamless and secure user experience.

Features:

Create and manage fundraising campaigns on Ethereum
Donate ETH to campaigns directly from your wallet
View all campaigns and your own created campaigns
Real-time updates and transparent donation tracking
Modern, responsive UI built with React and Tailwind CSS
Tech Stack:

Next.js (App Router)
React
ethers.js
Web3Modal
Solidity (Smart Contracts)
Tailwind CSS
Getting Started:

Clone the repository.
Install dependencies with npm install.
Start a local Ethereum node (e.g., Hardhat).
Deploy the smart contract and update the contract address/ABI in the frontend.
Run the app with npm run dev.
Connect your wallet (MetaMask) to interact with the platform.
